Windows清屏命令

2020-05-12

📎 PS：Windows下 == python , mysql == 清屏命令

python

version: 3.8.2

命令行内清屏：

import os
i = os.system('cls')

实现在 IDE 中任意输入后清屏：

import os
input()
os.system('cls)

可将其打包至自定义函数 自己调用

mysql

version: 8.0.19 ; Windows 10

实现命令行清屏
system cls
