练习 git 笔记：
// Git 学习笔记

  • git add 把文件传入暂存区
  • git status 查看仓库状态 、 掌握工作区状态
  • git commit -m ""从暂存区添加到仓库
  • git diff 查看 difference 、 查看修改内容
  • git reset --hard < commit_id > 在历史版本中穿梭
  • git log 查看提交历史
  • git reflog 查看历史命令
  • git checkout 丢弃工作区修改
  • git reset HEAD 将当前 HEAD 重置为指定状态
